This repository contains an end-to-end computational biology project investigating the molecular and cellular heterogeneity of **Juvenile Myelomonocytic Leukemia (JMML)**—a rare, aggressive clonal myeloproliferative disorder of early childhood. 

By integrating **bulk RNA-seq (global & FACS-sorted populations)**, **single-cell RNA-seq**, and **clinical microarray validation cohorts**, this project bridges macro-level transcriptomic shifts with single-cell hematopoietic stem/progenitor cell (HSPC) resolution.
